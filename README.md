# Crash Eagerly Patch (Java Edition)
In Java Edition, there are a bunch of debugging functions that are disabled by default. This mod enables the "Crash Eagerly" feature, allowing you to see one of Mojang's diagnostic tools.

**Warning:** Enabling the Crash Eagerly feature will cause the game to crash more often. Use at your own risk!

## Installation

To install this mod, place the JAR file in your "mods" directory. You must also install the Fabric API.

*Note: Although the patch can be used on multiplayer, it is strongly recommended that you do so in a separate testing environment.*
